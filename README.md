# Vanilla JS Challenge
### 바닐라JS로 크롬앱 만들기

##### 바닐라JS: 프레임워크와 라이브러리를 사용하지 않은 순수 자바스크립트
----
1. 로컬 스토리지를 이용한 로그인 구현
2. new Date()와 setInterval()을 활용하여 실시간 시간 나타내기
3. Math.random()을 이용한 랜덤 배경이미지 생성
4. 로컬 스토리지를 이용한 To-Do list(가고 싶은 여행지) 구현
